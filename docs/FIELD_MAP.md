# 附录A → App 字段映射

## 1. 总体原则

字段分为四层：

- L1 任务级字段
- L2 模板项字段
- L3 结果级字段
- L4 证据级字段

## 2. 编码规则

格式：

章节_分类_子项

例如：
- A1_BASIC_RES_NAME
- A2_CREST_ROAD
- A4_GATE_WORK
- A7_MONITOR_DEFORM
- A8_DOWN_BRIDGE

## 3. A.1 任务级字段

- A1_BASIC_RES_NAME: 水库名称
- A1_BASIC_DESC: 水库名称及基本情况描述
- A1_HUB_MAIN_STRUCTURES: 枢纽工程主要建筑物
- A1_FLOOD_PROTECT_OBJ: 水库防洪保护对象
- A1_CHECK_TIME: 检查时间
- A1_WEATHER: 天气
- A1_WATER_LEVEL: 检查时库水位/m
- A1_STORAGE: 检查时库容/m³
- A1_INSPECTORS: 检查人员
- A1_MAIN_PROBLEM_DESC: 现场检查发现的主要问题描述

建议扩展：
- A1_TASK_TYPE
- A1_DAM_TYPE
- A1_ROUTE_DESC
- A1_TOTAL_PHOTOS
- A1_TOTAL_ISSUES
