# 水库现场安全检查领域术语

项目领域：水库大坝现场安全检查

## 术语统一

- 挡水建筑物 = retaining structure
- 坝顶 = crest
- 坝体 = dam body
- 坝基 = foundation
- 坝肩 = abutment
- 上游坝面 = upstream face
- 下游坝面 = downstream face
- 上游坝坡 = upstream slope
- 下游坝坡 = downstream slope
- 防浪墙 = wave wall
- 反滤料 = filter material
- 排水设施 = drainage facilities
- 溢洪道 = spillway
- 输（引）水建筑物 = outlet / conveyance structure
- 溢（泄）洪隧洞 = spillway tunnel
- 检修闸门 = maintenance gate
- 工作闸门 = service gate
- 门槽 = gate slot
- 启闭机 = hoist
- 启闭房（塔） = hoist house / tower
- 挡污栅 / 拦污栅 = trash rack
- 消力池 = stilling basin
- 挑流鼻坎 = flip bucket
- 尾水渠道 = tailwater channel
- 水雨情测报设施 = hydrology telemetry facilities
- 安全监测设施 = dam safety monitoring facilities
- OMS = operation, maintenance and surveillance manual
